## Домашнее задание 1. Создание воспроизводимого ML-pipeline с использованием DVC и MLflow  
### Цели задания:  
Задание направлено на построение минимального, но полноценного MLOps-контура, обеспечивающего:  
- воспроизводимость экспериментов;  
- контроль версий данных и моделей;  
- автоматизацию процесса обучения и оценки модели;  
- документирование процесса через DVC и MLflow.  
  
### Инструкция по запуску DVC и MLflow
1. Выполните команду `git clone https://github.com/Lasiagrostis/mlops_hw1_kochedykov_vladimir.git`  
2. Откройте клонированный репозиторий `cd mlops_hw1_kochedykov_vladimir`  
3. Создайте виртуальную среду (venv/conda) и установите зависимости `pip install -r requirements.txt`  
4. Загрузите данные и кэш `dvc pull`  
5. Поднимите локльный MLflow командой `mlflow ui --backend-store-uri sqlite:///mlflow.db`. Активные эксперименты будут доступны по адресу http://127.0.0.1:5000  
6. Воспроизведите весь пайплайн `dvc repro`.  
  
### Описание пайплайна
`prepare.py` обрабатывает датасет **titanic**. На этом этапе происходит заполнение пропущенных значений, кодирование категориальных переменных и сохранение обучающей и валидационной выборки.  
Следующий шаг пайплайна - обучение модели DecisionTreeClassifier и запуск эксперимента в MLflow (скрипт `train.py`). Модель сохраняется в директории `/model/model.pkl`
